# SmartFlow Systems — Secrets Checklist

## Rule
Each app should only have the secrets it uses.

Shared where needed:
- SFS_PAT
- SFS_SYNC_URL
- OPENAI_API_KEY only if the app uses AI

Do not share app-specific secrets globally:
- DATABASE_URL
- SESSION_SECRET
- JWT_SECRET
- STRIPE_*
- GOOGLE_*
- X_*
- TWITTER_*
- SMTP_*
- SLACK_*
